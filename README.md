## Taras Devil Bot
### Features:
- Hate you
- Make jokes
- Show war stats
- Show weather forecast

## Setup
### Create & activate virtual environment
```
python3 -m venv "venv"
source venv/bin/activate
```


### Install dependencies
```
pip3 install -r requirements.txt
```

### Setup tokens in env
```
TELEGRAM_TOKEN
TELEGRAM_CHAT_ID

AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
AWS_REGION
```

## Start
```
python3 -m bot.py
```
